. build an image
+
```
podman build . -f ./ubi8/Dockerfile -t ubi8-check
# or 
podman build . -f ./ubi8-brew/Dockerfile -t ubi8-brew-check
```
+
. run the image
+
```
podman run -it --rm --entrypoint /bin/bash ubi8-check
# or 
podman run -it --rm --entrypoint /bin/bash ubi8-brew-check
```
+
